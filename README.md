## Pinouts
* D3 - I2S bitclock - RXD D3
* D19 - I2S word select - D19
* D18 - I2S data pin - D18
* D23 - Power

* D13 - Display data in
* D14 - Display clock
* D15 - Display CS
* D2 - Display DC
* D4 - Display RST

* D27 - VIN Temp
* D22 - SCL Temp
* D21 - SDA Temp
